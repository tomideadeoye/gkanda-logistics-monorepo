# NEXIM Port Terminal Presentation

This folder contains the presentation materials prepared for the Nigerian Export-Import Bank (NEXIM) showcasing the port terminal facilities.

## Contents

1. **NEXIM_Port_Terminal_Presentation.md** - Markdown version of the presentation
2. **NEXIM_Port_Terminal_Presentation.html** - HTML version with grid layout for images
3. **NEXIM_Port_Terminal_Presentation.pdf** - PDF version for easy sharing and printing
4. **1 - Terminal_Facility_Layout_Story.md** - Markdown version of the story-based presentation
5. **1 - Terminal_Facility_Layout_Story.html** - HTML version with enhanced visual presentation
6. **1 - Terminal_Facility_Layout_Story.pdf** - PDF version of the story-based presentation

## Presentation Overview

The presentations include:
- Terminal facility details including quay/wharf specifications
- Current terminal infrastructure with images displayed in a grid layout
- Completed security infrastructure information
- Technical compliance details
- Materials and durability specifications
- Quality assurance processes
- Maintenance and operations plan
- Project documentation summary

## Key Terminal Specifications

- **Quay Apron Area**: 37,305.84 m²
- **Quay Length**: 1,232.2 m
- **Quay Width**: 28.38 m

The presentations showcase images from `/images/terminal-b/` directory which demonstrate the current terminal facilities and completed infrastructure work, including a comprehensive layout image that tells the story of our facility development.

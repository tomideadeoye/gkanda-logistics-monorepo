### 📏 Technical Specifications
- **Fence Length**: ≈785 m (257 panels, 288 posts)
- **Height**: 2.44 m with 600 mm concertina topping
- **Posts**: SHS 60×60×3.0 mm S355 (line), larger at ends/corners
- **Foundations**: 350×350×900 mm C25/30 (line), larger at gates
- **Coatings**: HDG ≥85 μm + duplex DFT ≥160-200 μm (C5-M)
- **Wind Load**: V=30 m/s, net pressure p=331 N/m²

### 🎯 Standards Compliance
- Nigerian Building Code

- BS EN 1992/206/8500 (concrete)
- BS 1722 (chain-link fencing)
- ISO 1461 (hot-dip galvanizing)
- ISO 12944 (C5-M corrosion protection)

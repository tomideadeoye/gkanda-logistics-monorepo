## Overview

This package contains a comprehensive set of documents to support the development, implementation, and maintenance of an Environmental and Social Management System (ESMS) for GK and A Logistics Services Ltd. The package is specifically tailored to GK and A Logistics Services Ltd' operations as a maritime infrastructure and logistics development company with facilities in Ikorodu, Lagos.

The ESMS package consists of five main components:

1. **ESMS Implementation Handbook** - The core framework document (Enhanced Version)
2. **ESMS Toolkit** - Practical templates and forms for implementation
3. **ESMS Self-Assessment Guide** - Tool for evaluating system maturity
4. **ESMS Implementation Guide** - Step-by-step implementation instructions
5. **ESMS Policy Statement** - Official company policy document

## Setup and Usage

### Setting up Mermaid Support

To properly render diagrams in the documents, run the setup script:

```bash
./setup_mermaid_support.sh
```

This will install the necessary dependencies:
- Mermaid CLI for diagram generation
- Pandoc for document conversion
- Required Python packages

### Converting Documents with Diagram Support

1. **Option 1: Convert Mermaid diagrams to SVG images**
   ```bash
   python3 generate_md_with_svgs.py
   ```
   This creates a new Markdown file with SVG images instead of Mermaid code blocks.

2. **Option 2: Direct PDF conversion with Mermaid support**
   ```bash
   python3 convert_and_merge_v3.py
   ```
   This converts the Markdown file directly to PDF with embedded diagrams.

### Merging Documents

To create a complete handbook with cover page:
```bash
python3 convert_and_merge.py
```
or
```
python3 convert_and_merge_v2.py
```

## Troubleshooting

### Why am I seeing raw code instead of diagrams?

This is a common issue when previewing or converting Markdown files that contain Mermaid diagrams. The problem occurs because:

1. **Preview tools don't support Mermaid**: Many Markdown previewers don't have built-in Mermaid support
2. **Conversion scripts lack proper diagram rendering**: The conversion process doesn't wait for JavaScript to render diagrams
3. **Missing dependencies**: Required libraries for diagram rendering are not installed

### Solutions

1. **For preview**: Use an editor with Mermaid support (VS Code with Mermaid extensions)
2. **For PDF conversion**: Use the improved scripts with proper diagram support
3. **For reliable results**: Convert diagrams to SVG images first, then convert to PDF
